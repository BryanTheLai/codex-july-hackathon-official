# Recovery behavior

- A temporary Calendar availability failure stays in agent mode. The customer reply does not expose Calendar configuration or ask for staff intervention; it says that a time cannot yet be confirmed and requests one useful missing service detail.
- Schedule shows **Google Calendar needs connection** and provides an in-app connect/reconnect flow. The administrator enters the Calendar admin token for that request only; the UI clears it after obtaining the Google authorization link.
- A Telegram staff handoff shows its recorded reason. **Resume and reply now** re-enables autopilot and processes the latest eligible waiting customer message. If autopilot is already enabled but the reply has not completed, the action remains available as **Reply now**.
- New TTS replies are converted to OGG before storage and delivery. Local playback also converts legacy non-OGG artifacts when they are read, preventing the stale `0:00 / 0:00` browser duration caused by mislabeled audio bytes.
