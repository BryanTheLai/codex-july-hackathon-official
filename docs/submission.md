### Project description

KaunterAI is a Malaysia-first AI service desk for small aircon businesses. It
handles Malay and English text or voice requests, checks configured rates and
availability, books service visits, and turns customer corrections into
evaluated, human-approved SOP improvements.

### How Codex and GPT-5.6 were used

I used Codex with GPT-5.6 as a build-and-repair loop. I wrote the goals and
constraints in Markdown, then Codex implemented the work, ran tests, inspected
failures, repaired them, and repeated. The latest verified run passed 572 tests
across 82 test files.
