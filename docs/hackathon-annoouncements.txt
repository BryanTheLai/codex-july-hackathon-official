# Submission Format & Requirements

**Submission Deadline:** 18 July 2026 (Saturday) by 10:00 AM (MYT)

## Project Submission Steps

1. **Working Project:** A working project that meets challenge requirements, built with Codex using GPT-5.6, So/Terra/Luna.
2. **Category:** Continue working on your registered track.
3. **Project Description:** Explain what you created and how it works.
4. **Demo Video (<3 Min):** Upload a <3-minute public YouTube video demonstrating the project working, with audio covering your use of Codex and GPT-5.6.
5. **Code Repository:** Provide a URL to your code repository for judging and testing.
6. **Feedback Session ID:** Include your `/feedback` Codex Session ID where the majority of the core functionality was built.
* *How to get it:* Run `/feedback` in Codex, copy the Session ID, and paste it into the submission form.


7. **Additional for Plugins or Developer Tools:** If submitting a plugin or developer tool, include installation instructions, supported platforms, and a way for judges to test your project without rebuilding it from scratch (e.g., demo instance, sandbox, or test account).
